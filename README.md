# CodecTechnologies_Real-Time-Stock-Market-Dashboard

The Real-Time Stock Market Dashboard is an interactive data visualization application built using Python, Pandas, Plotly, and Streamlit. The project collects live stock market data through financial APIs and transforms it into meaningful visual insights for users, traders, and analysts.

This dashboard provides real-time price updates, historical trends, technical indicators, and dynamic visualizations that help users make data-driven investment decisions. With an intuitive UI, the system allows users to search for any stock symbol, track key metrics, and visualize market movement in seconds.
